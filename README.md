# MansyPlayer

MansyPlayer is an Android music player for people who keep their music locally and want a clean, focused listening experience.

It is built around local playback, audio tuning, lyrics, themes, playlist organization, and practical media tools. If you want a personal music player for your own files, MansyPlayer is ready to try.

## Download

The app is available on Google Play:

[Download MansyPlayer on Google Play](https://play.google.com/store/apps/details?id=com.mansy.music)

Install the Play Store version first. The hidden unlock puzzle below is designed for GitHub users who enjoy a small discovery path inside the app.

## Highlights

- Play and organize your local music library.
- Tune playback with equalizer and audio-effect features.
- View and manage lyrics for a richer listening experience.
- Customize the interface with themes and appearance options.
- Use playlist and music-space tools to keep your library organized.
- Explore local media utilities for everyday audio workflows.

## Premium Unlock Puzzle

### Entry Point

1. Install MansyPlayer from Google Play.
2. Open the app Settings page.
3. Find the "Version" row.
4. Tap it 5 times in a row to open the redeem-code input.
5. Enter the daily code you have decoded.

### Today's Ciphertext

```text
UGVhY2UgYW5kIExvdmUg
```

### Rules

Start from the ciphertext above and recover the original text used for the calculation.

After that, append today's date using this format:

```text
yyyyMMdd
```

Then calculate the MD5 hash of the full combined text. The last 6 characters of the hash are the daily code.

### Hints

- The ciphertext is not the final redeem code.
- The date uses the local date on the device running the app.
- Treat the MD5 result as the usual 32-character lowercase hexadecimal string.
- Code input is case-insensitive.
- Some characters may not be visible, but can still be part of the calculation.
